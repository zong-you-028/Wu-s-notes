# Definition

$$W = \int_1^2 F\cdot dx$$

Work is force acting through a displacement

# 正負號

$$
\left\{
	\begin{array}{}
		\text{ work done by a system } (+) \\
		\text{ work done to a system } (-)
	\end{array}
\right.
$$
