# Definition

$$ds = (\frac{ \delta Q }{ T })_{ rev }$$

$s$: Entropy

or

$$\delta Q = Tds$$

$$\Delta s = \underbrace{ s_2 - s_1 }_{ \text{ check table } } = \int_1^2\frac{ \delta Q }{ T }$$

# Entropy of a [[Pure Substance]]

Saturated:

$$s = (1 - x)s_f + xs_g$$
$$s = s_f + xs_{ fg }$$

2. Adiabatic Expansion
3. Isothermal Compression (Cooling)
