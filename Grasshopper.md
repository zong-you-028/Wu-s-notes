# 各分頁節點

## parameter

輸入變數或文字

- Panel: 輸入純文字

> ### 快速輸入
> 有兩種方法可以開啟輸入變數的文字方塊
> 
> - 按空白鍵
> - 按兩下左鍵

## display

用來輸出的

- text tag: 輸出平面文字
