# Laws

## Faraday's Law

$\phi_B$: 磁力線

$$v_L = N\frac{ d\phi_B }{ dt }$$

## Maxwell Circuit Law

$H$: 磁場強度

$$\oint Hdl = Ni$$

$B$: flux density
$\mu$: magnectic permenbility

$$H = \frac{ B }{ \mu }$$

$$B = \frac{ \phi_B }{ A\text{rea } }$$

$$\implies v_L = L\frac{ di }{ dt }$$

$$L = \frac{ N^2\mu A }{ l }$$

# Properties

$$v_L = L\frac{ di_L }{ dt }$$

## Energy Storage

$$E = \int Pdt = \int i_L v_L dt$$

$$ = \int_0^t i_L \cdot L \frac{ di_L }{ \bcancel{ dt } } \bcancel{ dt }$$
