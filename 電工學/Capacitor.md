Dielectric Constant $\epsilon$

$$\epsilon = \underbrace{ \epsilon_0 }_{ \text{ air } } \cdot \underbrace{ \epsilon_r }_{ \text{ relativity } }$$

![[Capacitor.png]]

$$v_c = \frac{ Q }{ C }$$

$C$: 電容 Capacitance (Unit: Faraday)

$$C = \epsilon\frac{ A }{ d }$$

# Properties

## 1. 

$$v_c = \frac{ Q }{ C }$$

$$ = \frac{ 1 }{ C }\int i_c dt$$

## 2. 

$$i_c = C \frac{ dv_c }{ dt }$$

## 3. Energy Storage

$$E = \int Pdt$$

$$ = \int i_c v_c dt$$

$$ = \frac{ 1 }{ 2 }C v_c^2$$
